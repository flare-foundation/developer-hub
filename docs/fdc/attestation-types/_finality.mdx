Blockchains have varying confirmation depths to consider blocks as final.

| Chain    | Confirmations required | Confirmation time |
| -------- | ---------------------- | ----------------- |
| Bitcoin  | 6                      | ≈60 mins          |
| Dogecoin | 60                     | ≈60 mins          |
| XRPL     | 3                      | ≈12 seconds       |
